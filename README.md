# Radio

A web-based radio player application built with Java and JavaScript.
![250611 1749586095 Desktop](https://github.com/user-attachments/assets/07de08d7-031d-4607-813c-07a169ef0b4c)

## Tech Stack

* Backend: Java, Maven, MySQL
* Frontend: JavaScript, HTML, CSS

## Installation

Prerequisites: Java JDK, Maven, MySQL Server

```bash
# Create database 'radio' in MySQL before running
git clone [https://github.com/yant0/Radio.git](https://github.com/yant0/Radio.git)
cd Radio
mvn clean install
java -jar target/*.jar
```

## Features

* Audio streaming playback
* Web-based user interface
* Volume and station controls
* Database storage for radio stations
* Login & register
* Like stations
